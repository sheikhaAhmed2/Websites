# 🍔 Burger Bliss Website

A fully functional, responsive burger joint website built with HTML, CSS, and JavaScript.

## Features

✨ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- Fully responsive layout with mobile-first approach
- Smooth scrolling navigation
- Touch-friendly menu

🎨 **Modern UI** - Clean and professional design
- Modern gradient backgrounds
- Smooth animations and transitions
- Professional color scheme

📱 **Interactive Elements** - Smooth user experience
- Smooth scroll navigation between sections
- Hover effects on menu items
- Intersection Observer for scroll animations
- Dynamic navbar shadow on scroll

🍽️ **Complete Menu** - All sections included
- Burger menu with 6 varieties
- Sides and drinks section
- About section
- Contact information

## Sections

1. **Navigation Bar** - Sticky navigation with links to all sections
2. **Hero Section** - Welcome banner with call-to-action button
3. **Menu** - 6 burger options with descriptions and prices
4. **Sides & Drinks** - French fries, onion rings, milkshakes, and soft drinks
5. **About** - Information about the burger joint
6. **Contact** - Location, phone, email, and hours
7. **Footer** - Copyright information

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with responsive design
- `script.js` - Interactive features and animations

## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the menu
3. Experience smooth scrolling and animations
4. Check out the burger menu and contact information

## Customization

You can easily customize:
- Colors: Edit the `:root` CSS variables in `styles.css`
- Menu items: Add/remove items in the menu grid in `index.html`
- Contact info: Update phone, email, address in `index.html`
- Business hours: Modify in the Contact section
